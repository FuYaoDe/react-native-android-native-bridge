# React Native Bridge 入門 Android 篇

> lastUpdate @ 20180201

## 前言

以下將介紹兩種作法：

- react-native new-library
- react-native-create-library

## 目標

實作一個 Nativ module 可以呼叫 Android 內建的 Toast 元件，以便顯示文字訊息。

## 環境準備

- Android Studio @ 3.0+
- JDK @ 1.8+
- Node.js @ 8.9.x
- React Native @ 0.52.2
- React Native CLI
- yarn
